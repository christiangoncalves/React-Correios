# React Correios

Project developed to consult packages in the Correios system.
Used Node.js with query plug-in in the Correios API.

# Requirements

- Git
- Node.js

# Execution

#### For Back End boot.
```
$node node-tracking/index.js
```

#### For Front End boot.
```
$cd react-tracking_
$npm run start
```
